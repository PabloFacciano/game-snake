# game-snake
Small snake game using Visual Basic Net (VB.Net)

## How to
To play, download repo and go to "src > Snake Game > bin > debug" and open `SnakeGame.exe`.

Arrows:
- Up and Space key: velocity
- Left and Right: Turn direction
